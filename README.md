  Data Science MSc student at FER (Zagreb). I build ML/NLP systems end-to-end — from data pipelines and evaluation to clean, reproducible code.

## What I’m working on
- **Spaniverse** — benchmark + tooling for evaluating LLMs on **span-level tasks** (e.g., entity linking).  
  Current focus: **knowledge base workflow** (Wikidata/Wikipedia builders → validated intermediate format → disk-backed KB) and evaluation plumbing.
- **Carefree** — a student–psychologist matching platform with an **LLM-based assistant**.  
  Building with real constraints in mind: safety, privacy, and human-in-the-loop usage, aiming for pilot-style testing in a counseling context.

## Research direction
- **LLM evaluation** that is reproducible (data, metrics, baselines, ablations).
- **Interpretability & robustness** in sensitive domains (health / education / high-stakes settings).
- Practical systems work: data → models → deployment-ready structure.
